# widya-betest
Assessment Techincal Test for Code ID

## Installation

1. Clone the project

```bash
  git clone https://github.com/widz11/widya-test.git ms-widya-betest
```

2. Go to the project directory

```bash
  cd ms-widya-betest
```

3. Setup env

```bash
  cp enc-example .env
```

4. Install dependencies

```bash
  npm install
```

5. Run migration

```bash
  npm run migration:run
```

6. Start the server

```bash
  npm run start
```

7. Go to service

```bash
  http://localhost:3030
```

8. Run test

```bash
  npm run test
```
